### Dependencies
- **cmake** minimum 3.14

### Instructions
**Linux**

1.  Download & install dependencies
3.  `$ cd malg`
4.  `$ mkdir build`
5.  `$ cd build`
6.  `$ cmake -G "Unix Makefiles" ..`
7.  `$ make`
8.  `$ ./test/runtest`

**Windows**

Note: The project has not been tested on Windows, but has been coded with portability in mind.
